# LLM Powered Fraud Detection System

A production-deployed fraud detection dashboard that analyzes transaction patterns, detects anomalies, and provides AI-driven risk insights.

Live Application:  
https://llm-fraud-detection-production.up.railway.app

---

## Overview

Financial fraud leads to billions in losses every year.  
This project demonstrates how intelligent analytics combined with AI-based insights can help identify suspicious transactions early.

The system provides a centralized dashboard for monitoring transactions, fraud counts, risk scores, and alerts.

---

## Features

- Interactive fraud analytics dashboard  
- Risk score visualization  
- AI-generated transaction insights  
- Alert monitoring system  
- Deployed on Railway  

---

## Tech Stack

Backend: Node.js, Express  
Frontend: EJS, Chart.js  
Deployment: Railway  
Version Control: Git & GitHub
